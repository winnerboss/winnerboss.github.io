60. 第k个排列

给出集合 [1,2,3,…,n]，其所有元素共有 n! 种排列。

按大小顺序列出所有排列情况，并一一标记，当 n = 3 时, 所有排列如下：

  1. "123"
  2. "132"
  3. "213"
  4. "231"
  5. "312"
  6. "321"

给定 n 和 k，返回第 k 个排列。

说明：

  • 给定 n 的范围是 [1, 9]。
  • 给定 k 的范围是[1, n!]。

首先这道题可以使用前面用过的回溯法，按照排列变化的规律去模拟得出前k个数的排列。但是对于这种能直接看出规律的问题，依次去求出前面的每个排列的做法时间效率就会很低。通过对题意的分析，我们可以利用不同位置上数字出现的规律，来直接得到每一位上需要的数字。对于第一位的数字，如果已确定一个数字，那么通过剩余位置上数字的变化，我们可以得到(n-1)!个不同的排列。所以在1到n的数字中，第k/(n-1)!小的数字就是第一位上的数字。余数m=k%(n-1)!就可以用来当做新的k去求第二个位置上的数字。我们通过一个辅助数组可以记录已经使用了哪些数字，再通过上面的方法，依次对每位去选取剩余第m/(n-1)!小的数字，就可以得到最终的排列方式了。
